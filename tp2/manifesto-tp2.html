RESUMO

O objectivo do trabalho resume-se a criacao de um ficheiro turtle (.ttl) com base num ficheiro json dado pelo professor.

O primeiro passo foi a transformacao do ficheiro json para um ficheiro xml utilizando uma ferramenta online.

Depois foi analisado o documento xml e procedeu-se a expesificao do modele a ser representado. Para isto foi criado o documento Protege com a representacao das relacoes entre as varias entidades.

Depois foi criado um ficheiro xsl, sendo este utilizado para a transformacao do ficheiro xml original para o ficheiro turtle pretendido. Depois da transformacao, foram feitos alguns ajustes, para melhorar a legibilidade do ficheiro e a organizacao dos conteudos.
